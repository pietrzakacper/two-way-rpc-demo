# Two way RPC demo
---
Simple demo feauturing 2 way RPC communication between C++ backend and JS frontend (Electron).


### Install dependecies

#### > C++
- [libjson-rpc-cpp](https://github.com/cinemast/libjson-rpc-cpp)

#### > JS
In `frontend` dir:
```
npm install
```

### Run
In `frontend` dir:
```
BACKEND_BIN_PATH=/path/to/backend_build npm start
```
