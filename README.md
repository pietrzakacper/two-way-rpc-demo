# Two way RPC demo
---
Simple demo feauturing 2 way RPC communication between C++ backend and JS frontend


### Install dependecies
- [libjson-rpc-cpp](https://github.com/cinemast/libjson-rpc-cpp)

### Build and run

```
cd frontend
npm i
cd ..
BACKEND_BIN_PATH=/path/to/backend_build node frontend/index.js
```
